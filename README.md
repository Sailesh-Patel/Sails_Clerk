## Deployment

To deploy this project, creat your own docker container by running:

```bash
  docker compose -t sails_clerk .
```


## Installation

Alternatively you can install install the front end with npm

```bash
  npm install 
  cd frontend
  npm start
```

And install the back end with mvn

```bash
  mvn clean install 

```
